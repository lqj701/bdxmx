### 2. 添加试卷接口
| 接口地址     | /api/paper/add |
| ------------ | ---------------------------------- |
| **请求方法** | POST                    |
| **详细说明** | 添加试卷接口 |

#### header参数说明
| 参数名称      | 类型    | 必选  | 描述       | 示例 |
| ------------- | ------- | ----- | ---------- | ---- |
| userToken   | String  | true |  userToken |   6405c8cc7b00ede85f7f1e123517e1b0   |


#### body参数说明--json
``` json
{
    "paperType":1, //试卷类型1：语文；2：数学；3：英语；4：物理；5：化学；6：生物；7：政治；8：历史；9：地理
    "name":"月考试卷一",
    "questionIds":[1,2,3,4],
    "totalScore": 100
}
```

#### 接口响应
```
{
  "code": 0,
  "message": "success"
}
 
```
