### 2. 审核
| 接口地址     | /api/user/review |
| ------------ | ---------------------------------- |
| **请求方法** | POST                    |
| **详细说明** | 绑定老师 |

#### body参数说明--json
``` json
{
    "studentIds":[1,2]
}
```

#### 接口响应
```
{
  "code": 0,
  "message": "success"
}
 
```
