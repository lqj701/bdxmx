### 1. 注册接口
| 接口地址     | /api/user/register |
| ------------ | ---------------------------------- |
| **请求方法** | POST                    |
| **详细说明** | 注册接口 |


#### body参数说明--json
```json
{
    "phone":15261851954,
    "password":"12345",
    "name":"abc",
    "email":"abcde@126.com",
    "avatarUrl":"",
    "type": 1,
    "gendar":false /false：男；true：女
}
```

#### 接口响应
```
{
  "code": 0,
  "message": "success"
}
 
```
