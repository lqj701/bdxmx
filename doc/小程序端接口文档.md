### 2. 获取所有老师
| 接口地址     | /api/mp/getTeacherList |
| ------------ | ---------------------------------- |
| **请求方法** | POST                    |
| **详细说明** | 获取所有老师 |


#### body参数说明--json
```json
{
    "page":1,
    "row":20
   
}
```

### 3. 绑定老师
| 接口地址     | /api/mp/bindTeacher |
| ------------ | ---------------------------------- |
| **请求方法** | POST                    |
| **详细说明** | 绑定老师 |

#### body参数说明--json
``` json
{
    "studentId":1,
    "teacherIds":[1,2]
}
```

#### 接口响应
```
{
  "code": 0,
  "message": "success"
}
 
```
