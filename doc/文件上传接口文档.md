### 1. 文件上传接口
| 接口地址     | /api/file/multiFileUpload |
| ------------ | ---------------------------------- |
| **请求方法** | POST                    |
| **详细说明** | 文件上传接口 |

#### body参数--form表单
| 参数名称      | 类型    | 必选  | 描述       | 示例 |
| ------------- | ------- | ----- | ---------- | ---- |
| file   | String  | true |  文件 |   1.jpg   |


#### 接口响应
```
{
  "code": 0,
  "message": "success",
  "data": [
    {
      "name": "f564dd5fb1f59482156a03c6a6e5e03.png",
      "url": "files.bdxmx.com/data/files/bdxmx/miniprogram/imagesf564dd5fb1f59482156a03c6a6e5e03.png"
    }
  ]
}
 
```
